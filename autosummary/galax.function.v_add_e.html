galax.function.v_add_e

galax.function.v_add_e(lhs_field, rhs_field, out)

Builtin message function that computes a message on an edge by performing element-wise add between features of destination and edge if the features have the same shape; otherwise, it first broadcasts the features to a new shape and performs the element-wise operation.

Broadcasting follows NumPy semantics. Please see https://docs.scipy.org/doc/numpy/user/basics.broadcasting.html for more details about the NumPy broadcasting semantics.

Parameters

  • lhs_field (str) – The feature field of destination.

  • rhs_field (str) – The feature field of edge.

  • out (str) – The output message field.
